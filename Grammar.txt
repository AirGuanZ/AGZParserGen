
AGZParserGen本身所使用的用来描述文法的脚本语法

IDENTIFIER：字母或下划线打头，后面跟零个或多个数字/字母/下划线
PATH：文件的相对路径，以'['开头、']'结尾

SCRIPT := STATEMENT*

STATEMENT := START_DEFINITION
STATEMENT := IMPORT
STATEMENT := RULE
STATEMENT := NAMESPACE

NAMESPACE := 'namespace' IDENTIFIER '{' SCRIPT '}'
START_DEFINITION := 'AGZ_Start' ':=' SYMBOL;
IMPORT := 'import' PATH
RULE := IDENTIFIER ':=' SYMBOL(+SYMBOL)*;

SYMBOL := IDENTIFIER(.IDENTIFIER)*
SYMBOL := TOKEN

TOKEN := 'token' '"' IDENTIFIER '"'
